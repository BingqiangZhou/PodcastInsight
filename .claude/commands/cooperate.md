# Cooperation Commands

Additional collaboration commands for team coordination.

## Commands

### /task "description"
**Smart Role Selection & Execution**
- Auto-detects required expertise and assigns to best agent
- Example: `/task "Add GraphQL API support"`

### /status
**Team Progress Check**
- Shows all agents and current task status
- Points to blockers and suggests next steps

### /handoff "role"
**Manual Context Transfer**
- Transfers current context to named role
- Includes work-in-progress and notes
